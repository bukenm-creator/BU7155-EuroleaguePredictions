# BU7155-EuroleaguePredictions

This code uses Euroleague history data and tries to predict 2025-2026 season matches by using GMMM algorithm and then creates some betslips according to the low-moderate-high risk levels by using tree search algorithms. Also it gives the Accuracy, Precision, RMSE, ME, AUC, MAPE scores of the system by applying model to train and validation datasets. All data is taken from Euroleague's own api by using following code repo on GitHub

https://github.com/bsamot10/EuroleagueDataETL/tree/main
